# Powerpoint_Real_Time_Clock
在PPT的VBA中实现了一旦开始放映幻灯片就自动运行的数字时钟。
如果需要在第2页以外的幻灯片实现，需要在VBA第四部分中的页码改为需要实现的对应页码。
In PowerPoint VBA, a digital clock is implemented that automatically runs once the slideshow starts.
If you want it to appear on slides other than slide 2, you need to change the page number in Part 4 of the VBA code to the corresponding slide number where you want it to run.
